ID Card Generator – User Manual

Access the ID Card Generator Portal here: https://cdcksd.github.io/icardgenerator/

Overview

The ID Card Generator is a tool designed to create personalized identification cards for individuals or in bulk using CSV files. It supports live previews, high-resolution outputs, and seamless downloading of front and back designs.

Single ID Card Generation

  1. Enter Required Information:
    • Name: Full name of the individual.
    • Course: Name of the enrolled program.
    • Registration Number: Unique ID or roll number.
    • Joining Date: Date of admission.
    • Expiry Date: Validity end date of the ID.
  2. Upload Photograph:
    • Click the “Choose File” button to upload an image.
    • Important: Image must be cropped to a 1:1 aspect ratio (square).
    • Use high-resolution images for best results.
    • Accepted formats include: JPG, PNG, JPEG, etc.
  3. Select Centre: Choose from CADD, DZ, Synergy, or LW.
  4. Generate and Download:
    • Click “Generate ID Card” to preview.
    • Click “Download ID Card” to save it as a PDF.

Bulk ID Card Generation via CSV

  1. Download the Template: Click the “Get Template” button to download a sample CSV file.
  2. Prepare the CSV File:
    • Fill in fields such as Name, Course, Registration Number, Joining Date, Expiry Date, and Image File Path.
    • Ensure all images are high quality and cropped to 1:1 ratio.
  3. Upload the Completed CSV: Use the second “Choose File” button to upload the CSV.
  4. Generate PDF:
    • Click “Generate Bulk PDF” to create the cards.
    • Note: Recommended to process a maximum of 15 members at a time for best performance.

Best Practices

  • Keep all images in one folder and ensure paths are correct in the CSV.
  • Avoid special characters or spaces in filenames.
  • Verify all fields before generating cards.
  • Securely store and back up generated PDFs.
